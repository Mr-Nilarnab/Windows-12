# Windows 12 Web Concept — Copilot+ AI PC 🚀

A futuristic, fully interactive web-based simulation of what **Windows 12** could look like, heavily focused on AI integration, glassmorphism aesthetics, and a fluid user experience. This project runs entirely in the browser and features a custom window manager, functional simulated applications, and interactive UI elements.

## ✨ Key Features

* **Advanced Window Manager:** Fully draggable, resizable, minimizable, and maximizable windows with snap-assist zones.
* **Copilot+ AI Integration:** A simulated on-device AI assistant pane that responds to prompts, summarizes text in Notepad, and acts as a central OS hub.
* **Dynamic Visuals:** Custom CSS canvas backgrounds (Aurora and Particle effects), deep dark mode, and Fluent Design glassmorphism.
* **Widget Panel:** Real-time clock, simulated weather, quick settings toggles, live-updating task manager stats, and news feeds.
* **Interactive Desktop & Taskbar:** Working Start Menu, system tray notifications, desktop icon selection, and a custom right-click context menu.

## 📦 Simulated Applications Included

This OS concept comes packed with custom-built web apps:
* **🌐 Edge Browser:** Simulated Bing AI search and browser navigation.
* **📁 File Explorer:** Navigable sidebars and file grid layouts.
* **💻 Terminal:** A working command-line interface mimicking PowerShell 7 (try `help`, `neofetch`, `sysinfo`, `ping`).
* **📝 Notepad:** Text editor featuring simulated "AI Summarize" capabilities.
* **🎨 Paint:** A fully functional canvas drawing tool with custom colors, brush sizes, and tools.
* **🔢 Calculator:** Working calculator for standard arithmetic.
* **🎵 Music:** A media player UI with progress tracking and simulated spatial audio tracks.
* **📊 Task Manager:** Live-updating CPU/RAM/GPU usage monitoring.
* **⚙️ Settings:** Interactive UI toggles and personalization menus.
* **💬 Teams:** Simulated chat environment.
* **🗺️ Maps & 📅 Calendar:** Conceptual UI designs for productivity and navigation.

## 🛠️ Tech Stack

This project is built from scratch without any heavy frontend frameworks.
* **HTML5**
* **CSS3** (Custom variables, Flexbox/Grid, Animations, Backdrop-filters)
* **JavaScript (Vanilla)** (DOM manipulation, HTML5 Canvas API, Event handling)

## 🚀 How to Run Locally

Because this project uses vanilla web technologies, running it is incredibly simple. No package managers or build steps required.

   git clone [https://github.com/Mr-Nilarnab/Windows-12.git](https://github.com/Mr-Nilarnab/Windows-12.git)
